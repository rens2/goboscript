# Control Flow

## repeat n times

```goboscript
repeat n {
    # code
}
```

## repeat until condition

```goboscript
until condition {
    # code
}
```

## forever loop

```goboscript
forever {
    # code
}
```

## if

```goboscript
if condition {
    # code
}
```

## if else

```goboscript
if condition {
    # code
}
else {
    # code
}
```

## if elif

```goboscript
if condition {
    # code
}
elif condition {
    # code
}
```

### if elif else

```goboscript
if condition {
    # code
}
elif condition {
    # code
}
else {
    # code
}
```
